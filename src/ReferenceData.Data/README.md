# Norse.ReferenceData.Data

Canonical reference-data entities (`Region`, `CountryOrArea`), the `ReferenceDataDbContext`, and `CountryOrArea`'s `RegionAncestry` owned-JSON ancestor chain. Runtime library — referenced by the migrations service and, later, Mímir's serving layer.

Part of the [Norse Architecture](https://github.com/NorseArchitecture) platform.
